# Seedance 2.0 Prompt 写作规范

## 触发条件

- 用户要求"做个视频"、"生成视频"、"seedance"、"豆包视频"、"即梦视频"
- Agent 准备调 `xrtoken video create`（或任何使用 Seedance 模型的视频接口）
- Prompt 短于 50 字 / 缺主体 / 缺动作 / 含违规词时强制改写

适用模型：`volcengine-overseas/dreamina-seedance-2-0-260128`、`-fast-260128` 及未来同系列。

---

## 1. 结构模板（必须按此顺序）

```
[镜头类型+运镜]，[场景环境]，[角色画面位置+朝向]，
[角色动作+表情+对白交织]。
[光影描述]。[环境音/情绪]。
[画质锚词]。[约束词]。
```

默认末尾加：**禁止出现任何文字、字幕、标题、水印**（除非用户明确要求文字/字幕）。

---

## 2. 时间分段

总时长 5–15 秒，按内容复杂度选段数：

| 复杂度 | 段数 | 写法 |
|--------|------|------|
| 简单 / 静态 | 1 段 | `0-8 秒画面：…` |
| 有动作 / 情绪转折 | 2 段 | `0-5 秒画面：…` + `5-12 秒画面：…` |
| 多对白 | 每段一句对白，时间留够 | 同上拆段 |

---

## 3. 镜头与运镜（必写）

**机位选项：** 正面平视 / 过肩镜头 / 仰视 / 俯视 / 侧面跟拍 / 主观镜头

**运镜必须减速：**

| ❌ 不要写 | ✅ 改写 |
|----------|--------|
| 快速推镜 | 镜头缓慢推进 |
| 甩镜 / 快切 | 镜头缓慢平移 |
| 急拉 | 镜头缓缓拉远 |

---

## 4. 角色空间调度（每镜必写）

| 维度 | 选项 |
|------|------|
| 位置 | 画面左侧 / 右侧 / 中央 / 前景 / 背景 |
| 朝向 | 面朝镜头 / 背对镜头 / 侧身面向 / 四分之三角度 |
| 视线 | 看向画面左侧 / 低垂 / 望向远方 |

---

## 5. 动作减速（绝对规则）

**严禁词：** 快速、猛烈、剧烈、迅速、突然、猛地

**改写表：**

| 原始 | 改写 |
|------|------|
| 快速转身 | 缓缓转身 |
| 奔跑 | 快步走 |
| 摔倒 | 身体缓缓倾斜 |
| 大笑 | 嘴角上扬 |
| 拥抱 | 轻轻靠近，手臂环住 |

**安全动作池：** 走路、站立、坐下、缓慢转头、微笑、皱眉、拿东西、望向某方向

**修饰词池：** 缓慢、轻柔、连贯、自然、流畅、缓缓、轻轻、微微

> 模型对快速动作容易穿模、抖动；动作越慢越克制，画面稳定度越高。

---

## 6. 情绪 → 光影映射

| 情绪 | 光影描述 |
|------|---------|
| 温馨 | 暖黄色柔光，侧面光源 |
| 紧张 | 冷色调，明暗对比强烈 |
| 悲伤 | 灰蓝色调，逆光 |
| 希望 | 清晨金色光线，正面柔光 |
| 日常 | 均匀自然日光 |

---

## 7. 画质锚词（倒数第二句）

- **写实**：`电影级画面，自然光线，写实影视质感，浅景深，电影摄影`
- **动漫**：`动漫风格画面，赛璐璐色彩，流畅线条，日系动画质感`

---

## 8. 约束词（最后一句，**含人脸时必加**）

```
面部稳定不变形，五官清晰，人体结构正常，比例自然，动作自然流畅不僵硬。
禁止出现任何文字、字幕、标题、水印。
```

---

## 9. 输出格式

直接输出 prompt 纯文本，**不解释、不加标题、不加 markdown**，中文 300–500 字。

---

## 完整示例

```
中景，镜头缓慢推进，教室走廊内景，自然日光从窗户照入。
画面左侧，一名 17 岁中国少年面朝镜头站立，身穿蓝白校服，表情腼腆略带紧张，双手不自然地攥着书包带。
0-5 秒画面：少年站在走廊中，眼神低垂，嘴唇微动，阳光从侧面打在他脸上。
5-13 秒画面：少年缓缓抬头，目光望向走廊尽头，脸上浮现一丝不确定的微笑。镜头缓缓拉近至近景，阳光在他脸上投下窗棂阴影。
暖黄色柔光，侧面光源。电影级画面，自然光线，浅景深。
面部稳定不变形，五官清晰，人体结构正常，比例自然，动作自然流畅不僵硬。
禁止出现任何文字、字幕、标题、水印。
```

## 多模态参考

如有图片/视频/音频参考素材，按**上传顺序**用 `图片1` / `图片2` / `视频1` 精确指代：

- "参考`图片1`中的人物形象"
- "动作参考`视频1`"
- "画面参考`图片2`的构图"

字幕、气泡台词、视频编辑、轨道补齐等场景的官方公式见 `references/seedance-scenarios.md`。

## Agent 改写流程（必读）

调 video create 前，按这个 checklist 走：

1. **判断要不要改写**
   - 用户 prompt < 50 字 / 没动词 / 含 "做个视频" 这种空话 → 必须改写
   - 用户 prompt 已具备 §1 结构 → 微调即可，不要画蛇添足
2. **填 §1 模板每一段**：镜头 → 场景 → 角色调度 → 动作 → 光影 → 画质 → 约束
3. **过 §5 动作减速表**：把所有快速动词替换；不在表里的快速词用修饰词池软化
4. **含人脸时**：必加 §8 约束词
5. **有参考素材**：用 `图片n`/`视频n` 显式指代，不要写"这张图/那个视频"
6. **写完通读一遍**，确保 300-500 字、纯文本、不带 markdown
7. **保留 NL 参数关键词**：prompt 里的"5秒/竖屏/1080p/不要声音"会被 xrtoken wrapper 自动提取为 CLI flag，不要把这些关键词删掉

## 反例 → 正例

❌ **用户原话直传：**
> 做个视频，主角跑过来抱住女主

❌ **问题：** 缺镜头、场景、光影、约束；动作"跑"和"抱"违反减速规则。

✅ **改写后：**
> 中景，镜头缓慢推进，傍晚的城市街道，路灯渐次亮起。
> 画面右侧，一名青年男子四分之三角度面向镜头，脚步快步走向画面中央的女子。
> 0-5 秒画面：男子快步走，面带笑意，女子背对镜头转身，神情温和。
> 5-12 秒画面：男子轻轻靠近女子，手臂环住对方肩膀，女子缓缓抬头看向男子。
> 暖黄色柔光，侧面光源。电影级画面，自然光线，浅景深。
> 面部稳定不变形，五官清晰，人体结构正常，比例自然，动作自然流畅不僵硬。
> 禁止出现任何文字、字幕、标题、水印。
